## 🚀 Featured Projects

### 🎙️ MeetMind AI — Real-Time Meeting Intelligence Platform

A real-time AI meeting intelligence platform that captures meeting audio, generates live transcripts, identifies speakers, and extracts structured insights.

**Key Features:**

* Real-time speech transcription using Whisper-based models
* Speaker diarization using pyannote.audio
* AI-powered extraction of action items, decisions, and risks
* Context-aware rolling meeting summaries
* Semantic similarity-based deduplication
* Entity resolution and confidence scoring
* Integration with Groq Whisper, NVIDIA Nemotron, and Google Gemini

**Tech:** Python · NLP · LLMs · Whisper · pyannote.audio · Gemini · NVIDIA Nemotron

[View Repository](https://github.com/Sopan777/MeetMind-AI)

---

### 🌿 AyurHealth — AI-Powered Ayurvedic Diet Management SaaS

An AI-assisted platform designed to help Ayurvedic practitioners create personalized diet recommendations using Prakriti assessment, nutritional data, and recommendation algorithms.

**Key Features:**

* Prakriti assessment for Vata, Pitta, and Kapha estimation
* ML-assisted constitution prediction
* Rule-based personalized food recommendation engine
* Integration of Ayurvedic principles with nutritional analysis
* Explainable hybrid recommendation architecture
* Practitioner feedback system for future model improvement

**Tech:** Python · Machine Learning · Recommendation Systems · Data Analysis · AI

[View Repository](https://github.com/Sopan777/ayur-wise-health)
